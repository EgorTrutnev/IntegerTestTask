Домашняя страница

Здравствуйте и Добро Пожаловать в небольшое single-page application, которое использует следующие технологии:

  • ASP.NET Core и C# с кросс-платформенным серверным кодом.
  • Angular и TypeScript для реализации пользовательского интерфейса.
  • Bootstrap 4 для стилизации пользовательского интерфеса.

В рамках данного проекта реализованы решения по техническому заданию. Сайт состоит из слудющих страниц:

  • Блок схемы поиска чёрных шаров в коробке с белыми. Для просмотра выберите пункт подраздела "Блок-схемы" в разделе "Коробка с шарами" навигационного окна.
  • Реализация алгоритма по поиску чёрных шаров в коробке с белыми. Для просмотра выберите пункт подраздела "Пример работы" в разделе "Коробка с шарами" навигационного окна.
  • Выборка данных, подходящих под определённые условия из базы . Для просмотра выберите пункт подраздела "Выборка из таблиц" в разделе "Работа с таблицами" навигационного окна.
  • Изменение/удаление/создание данных в базе. Для просмотра выберите пункт подраздела "Редактрирование" в разделе "Работа с таблицами" навигационного окна.

Для более полной информации о тех. задании перейдите в раздел "Тех.задания" в навигационном окне. Чтобы узнать об изменениях, внесённых после релиза проекта (версия 1.0), воспользуйтесь разделом GitHub в навигационном окне
