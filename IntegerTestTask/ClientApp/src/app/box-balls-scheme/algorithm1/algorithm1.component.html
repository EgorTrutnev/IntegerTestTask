Начало
На данном этапе осуществляется вход в программу.
Берём коробку с шарами
Основная задача на этом этапе сформировать коробку с шарами, поместив в неё один чёрный шар. Так как в коробке будет всего один чёрный шар, а все остальные - белые, достаточно просто создать переменную, в которую поместить позицию чёрного шара в коробке.
Достаём шар из коробки
Для решения задачи по изъятию шарика из коробки необходимо воспользоваться циклом, в котором будут поочерёдно (начиная с первого) доставать шары друг за другом.
Шар чёрный?
Для решения этой задачи необходимо использовать условный оператор сравнения. Здесь нужно проверить совпадает ли номер изъятого из коробки шар с номером чёрного шара. Как только мы получили чёрный шар в дольнейшем выполнеии цикла нет необходимости. Его нужно завершить. В C# для этого используется опертор break;.
Конец
На данном этапе осуществляется выход из программы.
