Начало
На данном этапе осуществляется вход в программу.
Берём коробку с шарами
Основная задача на этом этапе сформировать коробку с шарами. Сложность состоит в том, что общее количество шаров обоих цветов, шаров чёрного и белого цветов, а также расположение каждого из них нам неизвестно. Всё должно формироваться случайным образом. Для случайного определения количества чёрных шаров в коробке стоит ввсети понятие "концентрация".
Достаём шар из коробки
Для решения задачи по изъятию шарика из коробки необходимо воспользоваться циклом, в котором будут поочерёдно (начиная с первого) доставать шары друг за другом.
Шар чёрный?
Для решения этйо задачи необходимо использовать условный оператор сравнения. Здесь нужно проверить значение изъятого из коробки шара. Если оно равно "0" - шар белый, "1" - чёрный.
Забираем шар
Здесь можно выполнить любые задачи с полученным шаром. Но главное, нужно учесть есть, увеличив переменную, отвечающую за количество шаров на 1.
В корзине есть ещё шары?
На этом этапе необходимо вернуться к циклу из блока "Достаём шар из коробки", который, при наличии шаров, продолжит изъвлекать их из коробки, или остановится, при их отсутствии.
Конец
На данном этапе осуществляется выход в программу.
