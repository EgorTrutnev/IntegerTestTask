  • Разработать Web-пприложение CRUD на языке программрования C#, включающее реализцию 2 таблиц, имеющих реляционные связи.

    Предметная область - автопарк.

    • Таблица "Автомобили" содержит поля:

      Год выпуска Модель Гос. номер Дата постановки на учёт
      ... ... ... ...

    • Таблица "Тех. осмотры" содержит поля:

      Номер карты Дата тех. осмотра Замечания
      ... ... ...

    У одного автомобиля может быть несколько тех. осмотров.

    Приложение должно содержать следующие страницы (экранные формы):

    • Страница со списком автомобилей;
    • Страница добавления/редактирования сведений об автомобиле;
    • Страница со списком техосмотров;
    • Страница доавления/редактирования сведений о тех. осмотре;
    • Страница "Отчёт".

    Страница со списком автомобилей содержит кнопки: "добавить", "редактировать", "удалить", "отчёт". (Кнопки "редактировать" и "удалить" могут располагаться вверху списка или дублироваться для каждой записи).

    При нажати на кнопку "добавить" открывается страница добавления новой записи. Год выпуска должен выбираться из календаря.

    При нажатии на кнопку "редактировать" открывается страница редактирования записи, выбранной в списке.

    При нажатии на кнопку "отчёт", открывается страница, содержащая надписи с обобщенными (агрегированными) данными:

    • всего автомобилей: 'количество'
    • из них старше 3 лет: 'количество'
    • новее 3 лет: 'количество'

    Страница со списком тех. осмотров содержит кнопки: "добавить", "редактировать", "удалить".
